<h1 align="center">
Codigrate Color Palettes for CorelDRAW
</h1>

A carefully crafted collection of CorelDRAW color palettes inspired by nature and iconic cities around the world.
Each palette carries the signature colors of its Codigrate theme as a CorelDRAW custom palette, so the exact
same colors you code with become available everywhere you design.

## Getting Started

1. Download the **.xml** palette you want to use.
2. In CorelDRAW open **Window > Dockers > Palette Manager**.
3. Click the flyout menu (or the open-palette icon) and choose **Open Palette**, then pick the downloaded **.xml**.
4. The palette docks with all 24 named Codigrate colors, ready to use.

Or copy the **.xml** into **Documents\Corel\Corel Content\Palettes** and it appears under **Window > Color Palettes**.
